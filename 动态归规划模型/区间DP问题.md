集合：区间$ [i,j] $ 内的目标解

属性： Max、Min、数量 



状态计算：区间$ [i,j] $ 内的所有上一个状态的集合。即实现$[i,j]$ 的最后一次操作时的状态。



写法：先循环区间长度，再循环区间左端点，最后循环状态表示

```c++
for(int len = 0/1; len < n; ++len){
	for(int i = 1; i+len-1 <= n; ++i)
		for(状态枚举)
}
```
